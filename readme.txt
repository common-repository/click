Click is a plugin that will track vistor clicks on links.

== Description ==

This plugin adds javascript to the end of every web page.
It then tracks and records the links visitors click on the pages you want it to. 
You can then view where each visitor clicked to better enhance your website.
Click can be seen by time of day/week/month and what browser type made the click.

== Installation ==

1. Upload the `click` folder and each file in it to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to Settings in the dashboard, then Click Tracking and change the password at least.
1. Go to Manage Posts or Manage Pages and click Start to Start Tracking Posts and Pages.

== Frequently Asked Questions ==

= How can I view click tracking when logged out? =

You can view any page that is or was tracked by adding ?track= followed with your password from the Click Tracking settings to the end of the web page URL.

= Why are the dots over the wrong links? =

Due to different monitor sized, the dots are placed over links based on their index number. As such, if links are added it will mess up following links.
